# 📊 Data Analyst Securitisation Dashboard

## 📌 Project Overview

This project focuses on analyzing Auto Loan Securitisation data using Power BI. The dashboard provides insights into loan performance, delinquency trends, outstanding balances, vehicle-type distribution, and portfolio risk indicators.

The objective is to transform raw financial datasets into meaningful visual insights that support risk assessment and decision-making.

---

## 🎯 Objectives

- Analyze auto loan securitisation portfolio performance.
- Monitor delinquent loans and risk exposure.
- Evaluate outstanding balances across states.
- Visualize vehicle-type loan distribution.
- Create interactive dashboards for business insights.

---

## 📂 Dataset Used

The project uses the following datasets:

- Auto Loan Securitisation Data
- DPD Snapshot History
- Dynamic Loss Monthly
- Static Pool Vintage Data

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel

---

## 📈 Dashboard Features

### Key Performance Indicators (KPIs)
- Total Outstanding Balance
- Average Interest Rate
- Delinquent Loans
- Total Loan Portfolio

### Visualizations
- Vehicle Type Distribution
- Outstanding Balance by State
- Loan Delinquency Analysis
- Monthly Performance Trends
- Interactive Filters and Slicers

---

## 🔍 Key Insights

- Vehicle loan portfolios are distributed across multiple vehicle categories.
- Certain states contribute significantly to the outstanding balance.
- Delinquency patterns can be monitored through DPD analysis.
- Interactive filtering enables detailed portfolio exploration.
